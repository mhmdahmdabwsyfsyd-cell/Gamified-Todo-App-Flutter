# 🏆 Gamified To-Do App (Flutter)

A smart, interactive, and gamified To-Do List application built with **Flutter**. This app is designed to boost productivity by turning daily tasks into a fun game! Earn XP, level up, and unlock avatars as you complete your goals.

## ✨ Features

* 🎮 **Gamification System:** Earn **20 XP** for every completed task and level up for every 100 XP.
* ⏰ **Deadlines & Penalties:** Set deadlines for your tasks. If a task becomes overdue, you lose **10 XP** as a penalty!
* 🏷️ **Task Priorities:** Organize tasks by priority with visual color indicators:
    * 🔴 High Priority
    * 🟠 Medium Priority
    * 🟢 Low Priority
* 🛒 **Avatar Store:** Spend your hard-earned XP in the auto-scrolling store to unlock and equip cool profile pictures.
* 🌓 **Dark & Light Mode:** Beautiful UI with Glassmorphism effects, seamlessly switching between dark and light themes.
* 💾 **Local Storage:** Lightning-fast and secure offline data storage using **Hive**.
* 🎉 **Celebrations:** Enjoy a fun confetti animation and haptic feedback every time you complete a task!
* ⚡ **Smart Suggestions:** Quick-add buttons for common daily tasks.

## 🛠️ Technologies Used

* **Framework:** Flutter
* **State Management:** Provider
* **Local Database:** Hive (`hive_flutter`)
* **UI Effects:** `confetti` for celebrations, BackdropFilter for Glassmorphism.
* **Date Formatting:** `intl` package.
